This program takes in the numerical versions of numbers and writes them into their word form. 
   - Currently the ranges of numbers goes from -999999999999999 to 999999999999999

Author: Kevin Urban
Date last updated: 7/16/2017

-how to use:
   compile the program using the command:
      javac NumeralWriter.java
   Run the file using the command:
      java NumeralWriter

-Bugs:
   No known bugs in the program
